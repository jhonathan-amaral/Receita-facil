## Instalação

Clone esse repositório e instale as dependencias do projeto:

```bash
  npm install
```
  Comando para rodar o servidor 

```bash
  Rodar com IPV4: json-server --watch -d 180 --host Ip da sua maquina  db.json
```
Depois rode o projeto:

```bash
 npx expo start
```
